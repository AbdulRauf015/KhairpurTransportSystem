Khairpur Multi-Transport Management System
Developer: Abdul Rauf

SETUP STEPS:
1. Extract ZIP to E:\asmat\KhairpurTransport
2. Put mysql-connector-j-9.6.0.jar in lib\ folder
3. Open MySQL Workbench, run database\transport.sql
4. Double-click run.bat

LOGIN:
  Admin: admin / admin123
  User:  user1 / user123

FEATURES:
  Bus Booking, Car Booking, Bike Booking, Airline Booking
  Khairpur to Sukkur Airport transport + Flight booking
  Admin: Full CRUD, Reports, Print Passenger List
  User: Book any transport, view own bookings, print ticket
